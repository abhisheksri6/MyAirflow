Context Craft: Intelligent Document Vectorization Platform

AI-Powered Search and Metadata Enrichment for Unstructured Content

Why This Platform?

  • Enables fast, intelligent search across unstructured documents
  • Reduces manual data handling by over 70%
  • Built-in pipeline tracking and quality metrics
  • Flexible integration with existing systems
  • GenAI-ready for scalable use cases

Business Impact

  • Improved discovery through vector search
  • Faster insights with automated pipelines
  • Plug & play integration with your data sources
  • Compliance via document-level logging

Solution Architecture

  • Frontend: React UI for pipeline setup and control
  • API Layer: FastAPI handles UI interaction, routes to Airflow
  • Workflow Engine: Apache Airflow DAGs manage document processing
  • Vector DB: Qdrant stores chunk embeddings
  • Metrics DB: Postgres tracks logs and metrics

Component Overview

  • Frontend: React + ShadCN UI
  • API Layer: FastAPI for dynamic API generation
  • Workflow: Apache Airflow with parameterized DAGs
  • Storage: Qdrant for vectors, Postgres for metrics

Pipeline Workflow

  1. Import Documents
  2. File Type Detection
  3. OCR / Text Extraction
  4. Text Chunking
  5. Vectorization via Embedding Model
  6. Store in Vector DB
  7. Log to Postgres

Dynamic Search API Generation

  • APIs created based on form inputs
  • Supports top_k, filters, model config, etc.
  • Mounted via FastAPI routers dynamically
  • Accessible through status endpoint

Usage Tracking and Metrics

  • PostgreSQL logs each document and pipeline config
  • Pipeline name, parser, chunking, model
  • Processing time, success/failure status
  • Frontend displays usage stats and trends

Frontend Experience

  • Multi-step wizard for building pipelines
  • Search API creation tool
  • Live dashboard: active APIs, stats, logs

Deployment & Operations

  • Dockerized services: Airflow, FastAPI, Qdrant, Postgres
  • Airflow 3.0.2 with dynamic DAG creation
  • Runs in WSL2 / Docker Desktop
  • CI/CD ready

Roadmap

  • Short Term: more parsers, LLM-based extraction, retry logic
  • Mid Term: AuthN/AuthZ, DMS integrations, LLM QA interface
